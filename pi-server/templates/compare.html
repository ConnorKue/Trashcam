ACTIVE FILTER CONFIGURATION

Spatial Filter: Edge-preserving smoothing to reduce noise while maintaining object boundaries

Hole Filling: Fills missing depth data using surrounding valid pixels

  • Mode: Farthest from around (conservative approach)

Median Temporal Buffer: Median of last 10 depth readings

  • Prevents sudden spikes or drops in measurements
  • Provides stable fill percentage readings
